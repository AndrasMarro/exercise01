Verwendung einfacher Selektoren

Selektoren tun das, was ihr Name sagt: Inhalt in einer Webseite selektieren.

Sie vewenden eine CSS-artige Syntax:

  • : Selektiert alle Elemente mit Tag tag
  • : Selektiert alle Elemente mit Klasse className
  • : Selektiert alle Elemente mit id identifier
  • : Selektiert alle Elemente mit tag, die eine Klasse className besitzen
